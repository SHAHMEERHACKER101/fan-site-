# FanVue Creator Landing Page

Static referral landing page — deploy to GitHub + Vercel.

## Features

- Premium dark UI inspired by creator platforms
- Big **Sign Up** buttons → `https://www.fanvue.com/signup?referral=FV-LK1RC3`
- Auto-opens redirect link in new tab after **10 seconds**
- Full SEO: meta tags, Open Graph, JSON-LD, sitemap, robots.txt

## Local Preview

Open `index.html` in a browser, or run:

```bash
npx serve .
```

## Deploy

1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Replace `your-domain.vercel.app` in `index.html`, `robots.txt`, `sitemap.xml` with your live URL

See **AI-PROMPT.md** for the full AI generation prompt and SEO checklist.
